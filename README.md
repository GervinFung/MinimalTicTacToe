# Minimal Tic Tac Toe

## Game Features

| No. | Features                    |
| --- | --------------------------- |
| 1.  | Restart new game            |
| 2.  | Save game/ Load saved game  |
| 3.  | Show move history           |
| 4.  | Choose AI level from 1 to 4 |
| 5.  | Undo previous move          |
| 6.  | Choose grid size            |
